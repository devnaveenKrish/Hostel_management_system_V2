Project X

Project X, developed by college students, streamlines hostel booking and management, providing a practical application of web development skills.

Project X continues evolving, with students and faculty adding features, transforming it into a robust, scalable solution for college needs.

Life at Studio

At Project X, we prioritize efficiency and user-friendliness. We identify user requirements, use agile methodologies, and incorporate feedback to ensure each iteration meets high standards of functionality and usability.

Collaboration and innovation drive our workflow. Our team of students and faculty engage in brainstorming, code reviews, and testing, enhancing Project X with the latest technologies and best development practices.

Evolving Caspabilities

Project X displays various hostels around Lavasa, enabling users to choose their preferred hostel. The admin panel allows adding staff, users, and managing student requests efficiently.

The admin panel also features statistical graphs for analyzing hostel selections. Administrators can accept or reject student requests, providing a comprehensive and insightful management experience.
